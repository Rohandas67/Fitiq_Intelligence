# 💪 FitIQ — Health & Fitness Intelligence Platform

An end-to-end data science project combining health analytics
with machine learning to predict calorie burn and heart disease risk.

## 🚀 Features
- 📊 Interactive fitness data dashboard
- 🔥 ML-powered calorie burn predictor (R² ~0.99)
- ❤️ Heart disease risk analyzer (~85% accuracy)
- 📈 Full EDA with 10+ visualizations

## 🛠️ Tech Stack
Python | Pandas | Scikit-learn | Plotly | Streamlit | Joblib

## ▶️ How to Run
1. Clone this repo
2. pip install -r requirements.txt
3. Add datasets to data/ folder from Kaggle
4. Run notebook to generate models/
5. Run: streamlit run app.py

## 📁 Project Structure
FitIQ/
├── data/          # Add datasets from Kaggle 
├── models/        # Generated by running the notebook
├── notebook/      # Jupyter analysis notebook
├── app.py         # Streamlit web app
└── requirements.txt
